第一节导数的概念
定义 设函数在点x的某个邻域内有定义,当自变量x在x0处取得增量时.相应的函数值也会获得一定的增量,如果函数值的增量与自变量的增量之比在自变量增量趋近于零时极限存在, 那么称函数y=f(x)在点x处可导.并称这个极限为函数y=f(x)在点x处的导数.
根据函数在点x0处的导数f’(x)定义 导数是一个极限 而极限存在的充分必要条件时左右极限都存在且相等
第二节 函数的求导法则
函数的和差积商的求导法则
定理一 如果两函数点x处都有导数 那么他们的和差积商(分母为零的点除外)
相加求导等于分开求导再相加
相乘求导等于每次选一个求导 再相加
相除求导等于分子相减 分母取平方
反函数求导法则
定理二 如果函数x=f(y)在去间iy内单调.可导且f’(y)不等于零那么它的反函数
在去间ix内也可导 且反函数的导数等于直接函数导数的倒数
记录反正弦函数的导数
反余弦函数的导数
复合函数的求导法则
定理三 如果u=g(x)在点x可导 而y=f(u)在点u=g(x)可导那么复合函数在点x处可导 为 f’(u)*g’(x)

android

Share

android 系统架构大致可以分为四层架构 linux内核层 系统运行库 应用框架层 和应用层
linux内核层 android系统是基于linux内核的,这一层为android设备的各种硬件提供了底层的驱动.如显示驱动 音频驱动 照相机驱动 蓝牙驱动 wifi驱动 电源管理等等
系统运行库层 通过一些c/c++库来为android系统提供了主要的特性支持
应用框架内层 这一层主要提供了架构应用程序时可能用到的各种api (API（Application Programming Interface,应用程序编程接口）是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。)
应用层 所用安装在手机上的应用程序都是属于这一层
android 应用开发特色
android 系统由四大组件构成 分别是活动 服务 广播播放器 和内容提供器

九宫重排

Share

如下面第一个图的九宫格中，放着 1~8 的数字卡片，还有一个格子空着。与空格子相邻的格子中的卡片可以移动到空格中。经过若干次移动，可以形成第二个图所示的局面。

我们把第一个图的局面记为：12345678.
把第二个图的局面记为：123.46758
显然是按从上到下，从左到右的顺序记录数字，空格记为句点。
本题目的任务是已知九宫的初态和终态，求最少经过多少步的移动可以到达。如果无论多少步都无法到达，则输出-1。

数据结构简述

Share

链表 分散存储
每个数据元素都有一个指针指向它的直接后继
每个数据元素本身由存储数据信息的指针域，指向直接后继的指针域
结构体实现自定义
typedef stuct link
{
char elem；
struct link next； //代表指针域
}
链表的创建和遍历
link initlink（）{
link p = （link）malloc （sizeof（link））；
link temp=p；
for （int i=1；i<5 ;i++）{
link a =(link * )malloc (sizeof(link))
a->elem=i;
a->next=null;
temp->next = a;
temp=a;
}

return p;}

图的存储结构
多对多关系的数据 数据结构表示和存储这类数据的结构使用的是-图
顶点
使用图表示的每个数据元素称作顶点，途中所有顶点构成的集合习惯上用v表示。
有向图为弧 无向图为边
出度和入度
有向图中 顶点的度就是出度和入度的和
路径和回路
路径中每个顶点都不重复，这个路径就是简单路径，同样 一个回路中每个顶点都不重复这个路径就是简单回路。
连通图
在无向图中，如果一个顶点到另一个顶点存在至少一条路径，成他们之间是连通的，如果图中任意两个顶点之间都是连通的，则此图为连通的，如果一个图本身不是连通图，但是图中某个子图是连通的，那么子图被称为“连通分量”
生成树和生成森林
对于连通图来说，如果对其进行遍历，便利过程经过的顶点和边其实质是一棵树，在这里称其为生成树
生成树满足的要求
包含图中所有的顶点
任意两点之间有且仅有一条通路
连通图的生成树种，边的数量永远要比顶点的数量少一

图的顺序存储结构
数组表示法
使用数组存储图是，需要使用两个数组，一个数组存放图中顶点本身的数据，另一个数组用于存储个顶点之间的关系。
不同类型的图，存储方式略与不同，根据图有无权，可将图划分为两大类：图和网。
图用1 0表示 网用 权值 和 0 表示
图的链式存储结构
链式存储结构是实际应用中最常用的，使用链式存储结构表示图的方法有3种：邻接表，邻接多重表和十字链表

（1）r至少是个2位的2^k 进制数。
（2）作为2^k 进制数，除最后一位外，r的每一位严格小于它右边相邻的那一位。
（3）将r转换为2进制数q后，则q的总位数不超过w。
