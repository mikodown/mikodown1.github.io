蛇形矩阵

Share

蛇形矩阵是由1开始的自然数依次排列成的一个矩阵上三角形。

输入要求：
本题有多组数据，每组数据由一个正整数N组成。（N不大于100）
输出要求：
对于每一组数据，输出一个N行的蛇形矩阵。两组输出之间不要额外的空行。矩阵三角中同一行的数字用一个空格分开。行尾不要多余的空格。

样例输入：
5
样例输出：
1 3 6 10 15
2 5 9 14
4 8 13
7 12
11

代码实现：

#include<stdio.h>

#define maxn 105
int main ()
{
int xy[maxn][maxn];
int i,j,n,t=1;
scanf(“%d”,&n);
for ( i =0; i<=n-1;i++)

    if(i==0)
        for( j = 0; j<=n-1-i;j++)
            {    
            xy[i][j]= t;
            t=t+j+2;


        }
    else
        {
        for(j=    0; j<=n-i-1;j++)
            {
            xy[i][j]=xy[i-1][j+1]-1;
            }
        }    
for(i = 0; i<=n-1;i++)
    {for(j = 0 ; j<= n-i-2;j++)
    {
        printf("%d ",xy[i][j]);
    }
    printf("%d\n",xy[i][j]);
    }
return 0;

}
