c语言函数精要

Share

函数存在的意义
缩短代码长度；避免重复项操作
有利于程序的模块化（面向过程特点之一）

什么叫做函数
逻辑上：能够完成特定功能的独立的一个代码单元
物理上：能够将数据处理结果返回
能够对接受的数据进行处理
能够接受数据
总结： 函数是个工具，它是为了解决大量相似的问题而设计的
如何定义函数
函数的返回值，函数的名字，函数的形参列表 执行体
函数返回值的类型 以函数前的定义类型为准，如果和return类型不同，以函数前的数据类型为准

return 表达式和break的含义
    return 终止函数
    break 终止循环

函数分类
有参函数
无参函数
有返回值和无返回值
库函数和用户自定义函数
普通函数和主函数（main函数）
一个程序必须有且仅有一个主函数，主函数可以调用普通函数普通函数不能调用main函数，普通函数可以相互调用
注意的问题
普通函数要写道main函数前面
注意书写习惯 严格要求 声名形参类型
函数声明 可提前声明
形参和实参
如何在软件开发中合理的设计函数解决实际问题
一个函数的功能尽量独立，单一
多学习，多模仿牛人的代码
函数是c语言的基本单位，类是java c# c++ 的基本单位
常用的系统函数
double sqrt（double x）；//求x的平方根
abs（int x）；//求x绝对值
double fabs（double x）；//求绝对值
变量的作用域和存储方式：
按作用域分：
全局变量
所有在函数尾部定义的变量 都成为全局变量
适用范围 从定义位置到整个程序结束
局部变量
在一个函数内部定义的变量或者函数的形参 都称为局部变量 都称为局部变量
适用范围 函数内部使用
全局变量和局部变量冲突时：局部变量有效
按变量的存储方式
静态变量
自动变量
